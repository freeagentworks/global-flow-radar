### S-002: グローバル資金フローマップ（/map）

**目的**: 世界地図上に国・地域間の資金移動を可視化

**表示モード:**

1. **矢印モード**: 地域間に矢印を描画。太さ=資金量、色=方向（赤:流出、青:流入）
2. **バブルモード**: 各地域にバブルを表示。サイズ=純流入額、色=流入(緑)/流出(赤)
3. **ヒートマップモード**: 地域ごとに色分け（資金流入度合い）

**インタラクション:**
- 地域クリック → 詳細パネル表示（流入元/流出先の内訳）
- ホイール → ズーム（地域→国→市場レベル）
- 期間スライダー → 表示期間変更（1D / 1W / 1M / 3M / 6M / 1Y）
- アニメーション再生 → 時系列変化をアニメーションで表示

**データマッピング:**
```
地域ノード: US, JP, EU(DE+FR+UK), CN, EM(EEM), IN, BR, KR, TW, AU, Frontier
矢印データ: daily_flows テーブル WHERE flow_type = 'region'
```

**セクターローテーション判定ロジック:**
```

  - 後期拡大: エネルギー・素材への資金流入
  - 景気後退: 公益・生活必需品・ヘルスケアへの資金流入

表示: 現在の推定フェーズをハイライト
```

### S-004: 資産クラス間フロー（/asset-classes）

**目的**: 株式↔債券↔コモディティ↔暗号資産↔現金の資金移動

**メイン表示: サンキー図**
```
株式(米国) ──────┐
株式(国際) ──────┤
                ├──→ 債券(米国)
国債(長期) ──────┤──→ 債券(国際)
国債(短期) ──────┤──→ 金
                ├──→ コモディティ
金 ──────────────┤──→ 暗号資産
原油 ────────────┤──→ 現金等価
暗号資産 ────────┘
```

**サブチャート:**
- リスクオン/オフ指標メーター（VIX + クレジットスプレッド + 資金フロー方向の複合指標）
- 資産クラス別累積フローチャート（積み上げエリア）
- 相関マトリクス（資産クラス間の相関係数ヒートマップ）

### S-005: ETF/個別銘柄トラッカー（/etf-tracker）

**目的**: 個別ETFと銘柄レベルでの資金フロー追跡

**機能:**
- 検索バー: ティッカーシンボルで検索
- ウォッチリスト: お気に入りETFをピン留め（Phase 2: ログイン連携）

### S-006: ヒストリカル分析（/history）

**目的**: 過去の資金フローを遡って分析・比較

**機能:**
- 期間セレクター: カレンダーUIで自由期間選択
- イベントマーカー: 主要市場イベントをタイムライン上に表示
- 期間比較: 2つの期間を並べて比較（例: リーマン期 vs コロナ期）
- 再生モード: 期間指定でフローマップのアニメーション再生

**プリセットイベント:**

### S-007: アラート設定（/alerts）

**目的**: 急激な資金移動を検知して通知

**アラートルール設定UI:**
```
[新規アラート作成]
  対象タイプ: [地域 / セクター / 資産クラス / ETF]
  対象:       [ドロップダウン選択]
  条件:       [資金フロー > / < / 変化率 > ]
  閾値:       [数値入力]
  通知方法:   [アプリ内通知] ※MVP
```

**プリセットアラート（デフォルト有効）:**
- 全地域: 1日フロースコアが2σ超の変動
- 全セクター: 3日連続で同方向フロー加速
- VIX: 20%超の急騰

## 2. 共通UI仕様

### 2.1 ナビゲーション

サイドバー型（デスクトップ）/ ボトムナビ（モバイル）

```
📊 ダッシュボード   (/)
🗺️ フローマップ    (/map)
🔄 セクター        (/sectors)
📈 資産クラス      (/asset-classes)
📋 ETFトラッカー   (/etf-tracker)
📅 ヒストリカル    (/history)
🔔 アラート        (/alerts)
───────────
ℹ️ About           (/about)
```

### 2.2 レスポンシブ対応

| ブレークポイント | レイアウト |
|---|---|
| < 768px (モバイル) | シングルカラム、地図は横スクロール |
| 768-1024px (タブレット) | 2カラム |
| > 1024px (デスクトップ) | サイドバー + メインコンテンツ |

### 2.3 テーマ

- **ダークモード** をデフォルト（金融系アプリ標準）
- ライトモード切替可能
- カラーパレット: 
  - 流入: `#00C853` (緑系)
  - 流出: `#FF1744` (赤系)
  - ニュートラル: `#448AFF` (青系)
  - 背景: `#0D1117` (ダーク)

### 2.4 データ更新表示

- 最終更新日時を全ページヘッダーに表示
- データ取得中はスケルトンローダー表示
- 欠損データがある場合は該当箇所にツールチップで注記
