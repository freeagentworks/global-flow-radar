# 05. 開発フェーズ計画 - Global Flow Radar

## フェーズ概要

```
Phase 0 [2週間]    基盤構築・データパイプライン
Phase 1 [4週間]    MVP コア機能実装
Phase 2 [4週間]    MVP 完成・テスト・公開
Phase 3 [以降]     拡張機能・AI/ML統合・課金
────────────────────────────────────────────
合計 MVP: 約10週間（2.5ヶ月）
```

## リスクと対策

| リスク | 影響 | 対策 |
|---|---|---|
| Yahoo Finance API制限強化 | データ取得不可 | Alpha Vantage等のバックアップソース準備 |
| 資金フロー推計精度不足 | ユーザー信頼低下 | 信頼度スコア表示、複数指標クロス検証 |
| 1人開発でのスケジュール遅延 | MVP遅延 | AI支援最大化、機能優先順位の厳格化 |
| 無料枠超過（Supabase等） | コスト増 | データ保持期間の段階的縮小、キャッシュ強化 |
| 法的リスク（投資助言誤認） | 法的問題 | 免責表示の徹底、弁護士相談 |
