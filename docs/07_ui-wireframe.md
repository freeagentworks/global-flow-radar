### 2.5 コンポーネント仕様

#### FlowArrow（矢印コンポーネント）
```
Props:
  - source: { lat, lng }
  - target: { lat, lng }
  - flowScore: number (-100 ~ +100)
  - amount: number (USD millions)

表示ルール:
  - 矢印の太さ: |flowScore| に比例（1px ~ 8px）
  - 矢印の色: flowScore > 0 → #00C853, < 0 → #FF1744
  - 矢印の透明度: confidence値に連動（0.3 ~ 1.0）
  - アニメーション: ダッシュ配列で流れる方向を表現
```

#### MetricCard（指標カード）
```
Props:
  - label: string
  - value: number
  - change: number (%)
  - prefix: string ("$", "¥", "%")

表示ルール:
  - change > 0: 緑テキスト + ▲
  - change < 0: 赤テキスト + ▼
  - change = 0: グレーテキスト + ─
  - 数値: JetBrains Mono フォント
```

#### SectorHeatmapCell（セクターヒートマップセル）
```
Props:
  - sector: SectorData
  - metric: "flow" | "performance"

表示ルール:
  - 背景色: metric値に応じたグラデーション
    強い流入: #00C853 (100%) → #1B5E20 (50%)
    ニュートラル: #30363D
    強い流出: #B71C1C (50%) → #FF1744 (100%)
  - セル内テキスト: セクター名 + 値
  - ホバー: ツールチップで詳細表示
```

## 3. インタラクション仕様

### 3.1 ページ遷移
- サイドバーナビゲーション: 即時遷移（Next.js Link prefetch）
- ダッシュボード → 詳細: カード/ミニマップクリックで遷移
- 全遷移: ページトランジションアニメーション（fade 200ms）

### 3.2 データ更新
- 初期ロード: スケルトンローダー表示
- 期間切替: ローカルローディングインジケータ（チャート部分のみ）
- バックグラウンド更新: SWR/TanStack Query の stale-while-revalidate

### 3.3 ツールチップ
- チャート上ホバー: データポイントの詳細値
- 地図上ホバー: 地域名 + 純フロー額
- テーブル行ホバー: 行ハイライト

### 3.4 フィルター・ソート
- ドロップダウン: shadcn/ui Select
- 期間セレクター: ボタングループ（1W/1M/3M/6M/1Y）
- テーブルソート: カラムヘッダークリック（ASC/DESC）
- 検索: デバウンス付きインクリメンタル検索（300ms）

---

## 4. アクセシビリティ

- キーボードナビゲーション: Tab/Enter/Escape
- スクリーンリーダー: aria-label、chart alt text
- 色覚多様性: 色だけでなく矢印方向・アイコンで方向を表示
- フォーカスインジケーター: 全インタラクティブ要素に可視フォーカス
